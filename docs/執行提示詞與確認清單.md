### 來源文字檔 `pdf_dump.txt` 的重建方式

`pdf_dump.txt` 是從 PDF 擷取出的純文字，存放在工作目錄根目錄（非 `docs/`），
可能未被 git 版控。若在新電腦上找不到此檔，請先執行以下指令重新產生
（需要 `pip install pypdf`）：

```python
import pypdf
reader = pypdf.PdfReader(r'docs\Unitech_RFID_Command_Set_V1.02_20250814_User.pdf')
with open('pdf_dump.txt', 'w', encoding='utf-8') as f:
    for i, page in enumerate(reader.pages):
        text = page.extract_text() or ''
        f.write(f'--- Page {i+1} ---\n')
        f.write(text)
        f.write('\n\n')
```
