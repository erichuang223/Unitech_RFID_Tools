# 每日設計討論紀錄（Daily Log）

> 本檔案以 **daily（依日期）** 方式，記錄使用者明確陳述的設計決策與規劃內容。
> 用途：保留討論脈絡，避免後續產出時遺忘。
>
> ⚠️ **此處內容不等於規格**。規格內容由 Claude 收斂產生，存放於各階段的規格文件中，
> **不可用本日誌的原始討論內容直接汙染規格文件**。唯一例外：使用者提供的「圖」
> 作為參考依據時，該圖可被引用進規格文件。
